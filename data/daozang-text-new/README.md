# 道藏_txt

《正统道藏》及《续道藏》全文文本版

Complete text version of the Zhengtong Daozang (正统道藏) and Xu Daozang (续道藏)

## 简介

本仓库包含明正统年间编纂的《正统道藏》及万历年间续补的《续道藏》的完整文本版本，共计 **1504** 部经典，约 **64 MB** 文本数据。

## 目录结构

按"三洞四辅"分类：

### 三洞 (Three Caverns)
- **洞真部** (Dongzhen) - 上清经系: 319部
- **洞玄部** (Dongxuan) - 灵宝经系: 303部
- **洞神部** (Dongshen) - 三皇经系: 364部

### 四辅 (Four Supplements)
- **太平部** (Taiping): 65部
- **太玄部** (Taixuan): 113部
- **太清部** (Taiqing): 24部
- **正一部** (Zhengyi): 237部

### 续道藏
- **续道藏** (Xu Daozang): 59部

## 历史背景

《正统道藏》是明代第43代天师张宇初奉敕编纂，刊成于正统十年（1445年），共480函，5305卷。万历三十五年（1607年），第50代天师张国祥奉命续补《续道藏》180卷。这是现存最早的官修道藏，也是研究道教文献的重要资料。
